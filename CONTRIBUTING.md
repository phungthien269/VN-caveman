# Contributing

Improvements to the `gon` behavior, examples, and runtime surfaces are welcome.

## How

1. Edit the source-of-truth file first:
   - `skills/gon/SKILL.md` for main behavior
   - `skills/gon-commit/SKILL.md` for commit behavior
   - `skills/gon-review/SKILL.md` for review behavior
   - `gon-compress/` for compression runtime
   - `rules/gon-activate.md` for always-on rule text
2. Update matching docs and examples.
3. If behavior changed, update `DECISIONS.md` and `PROGRESS.md`.

Small, focused changes are preferred over broad rewrites.
