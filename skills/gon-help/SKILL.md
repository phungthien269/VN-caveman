Hiển thị quick reference. Không đổi mode.

## Modes

- `lite`: gọn vừa, câu đầy đủ
- `full`: mặc định, ngắn và trực diện
- `ultra`: cực ngắn, shorthand dev

## Skills

- `gon`: response compression
- `gon-commit`: commit message
- `gon-review`: code review
- `gon-compress`: nén prose/docs

## Safety

Tự tăng clarity khi:

- destructive op
- prod/infra/database
- security
- migration nhiều bước

## Off

- `stop gon`
- `normal mode`
- `trả lời bình thường`
